My goal is to make responsive, beaultiful and functional websites, using technologies such as HTML, CSS and JavaScript.

Hypertext Markup Language (HTML) is used to set the content of a webpage, be it a text, an image, or a video!.

The Cascading Style Sheets (CSS) were made to make HTML pages more appealing! With it, you can build the styles of your site.
